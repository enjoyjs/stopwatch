# stopwatch

A stopwatch utility

## Install

```bash
npm i @enjoyjs/stopwatch
```

Then import this:

```ts
import Stopwatch from '@enjoyjs/stopwatch';
```

## API

### class: Stopwatch

```ts
const stopwatch = new Stopwatch();
```

#### stopwatch.start()

#### stopwatch.pause()

#### stopwatch.reset()

#### stopwatch.isRunning()

#### stopwatch.getTime()
